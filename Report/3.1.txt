Resumo das alterações e testes

Alterações de código
- validation.c
  - `pwm_is_valid_user`:
    - Aceita comprimento máximo (<= PWM_MAX_USER_ID_LEN).
    - Itera desde o índice 0 para validar o primeiro carácter.
    - Corrige a comparação de letras minúsculas para incluir 'z' (`c > 'z'`).
  - `pwm_is_valid_password`:
    - Corrige checagem de comprimento para aceitar extremos ("< min" ou "> max").
    - Percorre toda a blacklist (começava em 1; agora começa em 0).
    - Itera apenas até `i < len` (antes incluía o terminador nulo).
    - Corrige verificação de dígitos (`&&` em vez de `||`).
    - Corrige lógica de pontuação: permite no máximo UMA pontuação dos
      caracteres ".:,!?" (segunda pontuação rejeita).
    - Caracteres inválidos causam `PWM_INVALID_PASSWORD` (antes devolviam OK).

Testes adicionados/alterados (gtest)
- gtest/testuser.cpp
  - Testes novos: `min_length_ok`, `max_length_ok`, `uppercase_invalid`,
    `invalid_middle_char`.
- gtest/testpass.cpp
  - Testes novos: `min_length_ok`, `max_length_ok_with_one_punct` (corrigido),
    `two_punct_invalid`, `missing_upper_invalid`, `missing_lower_invalid`,
    `missing_digit_invalid`, `invalid_char_space`.
  - Corrigi um teste que usava duas pontuações em vez de uma (ficou inválido).
